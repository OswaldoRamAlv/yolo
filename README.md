# YOLO
Ejercicio de prueba para aprender el funcionamiento del algoritmo YOLO de ultralitics, utilizando como ejemplo capturas de juegos de futbol americano para la deteccion de cascos

# Fuentes de datos
los datos utilizados en este ejercicio fueron obtenidos de Kaggle, de la competencia NFL 1st and Future - Impact Detection
https://www.kaggle.com/competitions/nfl-impact-detection
